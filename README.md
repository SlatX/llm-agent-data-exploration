# Retail Sales Analysis via LangChain Agents

This repository contains a Google Colab notebook demonstrating how to use LLMs to automate complex data science workflows. 

## Key Features
- **Autonomous EDA:** Used a LangChain Pandas Agent to query and clean 421,570 rows of data.
- **Statistical Significance:** Conducted ANOVA and T-tests to quantify the "Holiday Effect" on weekly sales.
- **Store Segmentation:** Applied K-Means Clustering to group stores into High, Mid, and Low performance tiers.
- **LLM Integration:** Configured for OpenRouter to utilize `gpt-4o-mini` for cost-effective, high-iteration reasoning.

## Tech Stack
- **Language:** Python 3.12
- **Frameworks:** LangChain, Scikit-Learn, Pandas, Matplotlib
- **LLM:** GPT-4o-mini (via OpenRouter)

## Data Source
The analysis is based on the Walmart Recruiting - Store Sales Forecasting dataset.
